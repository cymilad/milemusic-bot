### Prerequisites

* MySQL
* PHP
* Host & Domain


### Run Server

Go to Cpanel 🐧:

```bash
1 - select Manage My Databases
2 - Create New Database
3 - Database Users and Add New User/Password
4 - Add User To Database
5 - Then Back Go to PHPMyAdmin select Database
6 - import music.sql
7 - Edit config.php
8 - Run Bot
```
